## 已知問題（後端需修正）

| 問題 | 說明 |
|------|------|
| `remainingTickets` 未正確遞減 | 報名成功後票數沒扣，導致前端顯示「已額滿」但仍可報名 |
| Race condition | 兩人同時搶票，後端可能發出兩張 confirmed 票 |
| 孤兒票券（Unknown Event） | 刪除活動時未 cascade 刪除票券，導致票券頁出現 Unknown Event |
| 缺少 `PATCH /users/{userId}/lock` | 只有 unlock，沒有 lock，福委無法手動鎖定帳號 |
| `cancellationDeadline` 未回傳 | Transaction API 沒有帶此欄位，前端取消截止判斷失效 |
| HR 票券 API 權限 | `GET /tickets`、`POST /transactions` 只允許 employee，HR 被擋 |
